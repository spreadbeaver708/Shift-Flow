使い方

必要な操作だけを順番に説明します。

{{ back_link(url_for('menu') if current_user else url_for('login'), 'メニューに戻る' if current_user else 'ログイン画面に戻る') }}

1. ログイン

  1. 管理者から渡されたIDと一時パスワードを入力します。
  2. 初回は、15文字以上の新しいパスワードへ変更します。
  3. 何度も失敗した場合は、1分待ってからやり直します。
{% if not current_user or current_user.role == 'worker' %}

2. シフト希望を出す

  1. 「自分のシフト希望を入力」を開きます。
  2. 今月または翌月を選びます。
  3. 日・月・木を「〇 出勤可」か「× 休み」にします。
  4. 必要な日だけ備考を追加し、「この内容で保存」を押します。
備考に相談内容、健康情報、個人情報を書かないでください。

3. 確定シフトを見る

メニューの「確定シフトを見る」から、決まった出勤日を確認します。

{% endif %} {% if current_user and current_user.role == 'admin' %}

管理者の流れ

  1. 「提出状況を見る」で未提出者を確認します。
  2. 「希望シフトを見る」で希望と備考を確認します。
  3. 「確定シフトを作る」で職員ごとの出勤日を決めます。
  4. 月1回、操作ログとバックアップを確認します。

職員を追加・停止する

「ユーザー管理」で追加します。退職や休職は削除せず「停止」にします。

{% endif %}
