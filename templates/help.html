使い方

必要な操作だけを順番に説明します。

{{ back_link(url_for('menu') if current_user else url_for('login'), 'メニューに戻る' if current_user else 'ログイン画面に戻る') }}

1. ログイン

  1. 管理者から渡されたIDとパスワードを入れます。
  2. パスワードはメニューの「パスワード変更」でいつでも変えられます。
  3. 何度も失敗したときは、1分待ってからやり直します。
{% if not current_user or current_user.role == 'worker' %}

2. シフト希望を出す

  1. 「シフト希望を入力」を開きます。
  2. 今月または翌月を選びます。
  3. 日・月・木を「〇 出勤可」か「× 休み」にします。
  4. 必要な日だけ備考を入れて、「この内容で保存」を押します。
備考に相談内容や健康などの個人情報は書かないでください。

3. 締め切りについて

締め切り日になると、希望は変更できなくなります。直したいときは管理者に伝えてください。

{% endif %} {% if current_user and current_user.role == 'admin' %}

管理者の流れ

  1. 「提出状況・締め切り」で締め切り日を決め、未提出の人を確認します。
  2. 各職員の「編集」から、希望を直接なおせます（締め切り後も可）。
  3. 「みんなの希望を見る」で全員の予定をまとめて確認します。
  4. 月1回、操作ログとバックアップを確認します。

職員を追加・停止する

「ユーザー管理」で追加します。退職や休職は削除せず「停止」にします。
