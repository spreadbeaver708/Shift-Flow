## 📌 決定記録（変更する場合は理由ごと見直すこと）

| 決定 | 理由 | 再検討の条件 |
|---|---|---|
| `TRUST_CF_CONNECTING_IP=1`（Render） | Render は Cloudflare を前段に持ち、全インバウンドがエッジ経由のため CF ヘッダが実クライアントIP。公開情報で確認済み（2026-07-02レビュー §4） | Render 外へ移設するなら **必ず 0** に |
| `TRUSTED_HOSTS` 未設定（検証オフ） | 初心者がヘルスチェックごと締め出す事故を防ぐ安全側の既定 | 防御強化したいとき WO-08 |
| パスワード8文字・定期変更なし | NIST SP 800-63B は定期変更を非推奨。小規模運用の受容判断 | 機微情報を扱うなら 15文字/MFA を再評価（WO-05） |
| gunicorn 単一 worker + `memory://` | SQLite とメモリ内レート制限の前提。10名規模では十分 | worker を増やす前に WO-03 |
| `PYTHON_VERSION=3.14.5` 固定 | 動作再現性優先（GC挙動の安定） | パッチ追従したくなったら `3.14` 系指定へ（月次点検で判断） |
| 依存はピン留め・一括更新しない | 更新は月次 `pip-audit` の結果に基づき最小限 | 脆弱性が出たときのみ該当パッケージを更新 |
